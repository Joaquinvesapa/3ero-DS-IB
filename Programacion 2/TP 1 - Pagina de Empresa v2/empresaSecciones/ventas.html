Ventas

Nueva venta

Se podrá registrar una nueva venta realizada, incluyendo información como el cliente, los productos vendidos y su precio, el tipo de pago, entre otros.

Historial de ventas

Se podrá consultar todas las ventas registradas en el sistema, ordenadas por fecha o cliente, así como filtrarlas por rango de fechas.

Cotizaciones

Se podrán generar cotizaciones de productos para enviar a los clientes, incluyendo información como el precio, el tiempo de entrega, el tipo de pago, entre otros.

Facturación

Se podrá generar facturas de venta para los clientes, incluyendo información como el número de factura, los productos vendidos y su precio, el monto total, entre otros.
