Lista de productos

Se podrá consultar la lista completa de productos del inventario, incluyendo información como el nombre del producto, la descripción, el precio, entre otros.

Agregar producto

Se podrá agregar un nuevo producto al inventario, incluyendo información como el nombre del producto, la descripción, el precio, entre otros.

Editar Producto

Se podrá modificar la información de un producto ya existente en el inventario

Eliminar Producto

Se podrá eliminar un producto del inventario

Control de stock

se podrá llevar un registro detallado de los productos que se encuentran en el inventario, asi como también controlar la cantidad disponible de cada uno de ellos.
